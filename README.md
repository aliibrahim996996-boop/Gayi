# Présentation de Gayi — Site web

Site de présentation de la ville de **Gayi** (région de Zinder, Niger) — page HTML/CSS simple créée par **Ali Ibrahim Adamou**.

---

## Aperçu
Ce site présente la géographie, l'hydrogéologie, l'histoire et des informations locales sur Gayi. Conçu pour être léger et immédiat : un `index.html` + styles intégrés.
